## 起動方法

以下のようにしてサーバを起動

```bash
npm run dev
# or
yarn dev
```

[http://localhost:3000](http://localhost:3000)を開けば「Hello world」が出る。

## PrettierとESLint
この二つは併用して動かしている。
最低限の設定しか行っていないため、各自お好みで。

## Renovateの導入
Renovateを最低限動かすファイルは作っているので、
リポジトリ作成後、各自設定する。
いらない場合は、.github/renovate.jsonを削除する。
